# ANTON | AI Insight Engine

![Anton Dashboard](https://images.unsplash.com/photo-1551288049-bebda4e38f71?auto=format&fit=crop&w=1200&q=80)

Anton is a full-stack "Smart Data Analyzer" designed to extract intelligence from raw text and **PDF Documents**. Built with a high-performance FastAPI backend and a cutting-edge React frontend, Anton provides real-time sentiment analysis, reading metrics, and key phrase extraction.

## 🚀 Key Features
- **PDF Document Parsing:** Upload full PDF reports for instant AI extraction.
- **Deep Sentiment Analysis:** Real-time emotional scoring of text inputs.
- **NLP Extraction:** Automatically identifies core subjects and key noun phrases using TextBlob.
- **Velocity Tracking:** Interactive Recharts dashboard showing sentiment trends.
- **Report Generation:** Instant .txt report downloads for any analyzed entry.

## 🛠️ Tech Stack
- **Frontend:** React, Vite, Tailwind CSS v4, Recharts.
- **Backend:** Python, FastAPI, SQLAlchemy, PyPDF2.
- **Environment:** Designed for cloud deployment (Render/Vercel).
